This section illustrates the Spring container configurations through annotations and it covers the following: 
1- Inversion of Control IoC
2- Dependency Injections through @Autowired
  a- Constructor argument
  b- Setter methods
  c- Fields
3- Injecting literal values through a property file and @Value
4- Bean Scope and lifecycle
  a- Scope values @Scop
  b- Initialization and destroy methods @PostConstruct @PreDestroy
