This application is an update for the CRM application by adding security 
This app covers Spring MVC, Hibernate, Security , AOP
This section contains scripts for Mysql scheme (one for customer info and one for username and passwords)
Through this app you can register a new user (with Employee Role)
or login with the existing accounts
