This section covers Hibernate ORM through the following:
1- Define CRUD operations on a database
2- Table mapping:
  a- one to one
  b- one to many
  c- many to many
3- Data fetch types (eager and lazy)
4- All code examples are supported with sql scripts to create database schemes
5- The tables and their mapping is described as follows:
  Tables: Instructor- InstructorDetail- Course- Review- Student 
  
  Instructor- InstructorDetail (one to one)
  Instructor-Course (one to many)
  Review- Course (many to one)
  Course- Student (many to many)
